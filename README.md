# devops-toolbox

Reusable DevOps setup scripts, automation, and environment configurations. Satellite repo of [DevOps-LearnIT](https://github.com/ddeer1109/DevOps-LearnIT).

## Structure

| Folder | Purpose |
|--------|---------|
| `wsl/` | WSL setup & configuration scripts |
| `ubuntu-server/` | Ubuntu server provisioning |
| `dotfiles/` | Shell configs, aliases, tool configs |
| `scripts/` | Reusable automation scripts |
| `mcp-servers/` | MCP server configs for Claude Code |

## Usage

Clone alongside the hub repo:

```bash
git clone https://github.com/ddeer1109/DevOps-LearnIT.git
git clone https://github.com/ddeer1109/devops-toolbox.git
```
